# Arrays

Array fundamentals and operations.

## Files

- `ArraySum.java` - Sum of array elements with JOptionPane
- `ArraySort.java` - Array sorting
- `ArrayReverse3.java`, `ArrayReverse4.java`, `ArrayReversed.java` - Array reversal methods
- `ArrayCheckKeyRandom.java` - Key checking
- `ArrayCountKeyOccurences.java` - Counting occurrences
- `ArrayFortuneTeller.java` - Fortune teller using arrays
