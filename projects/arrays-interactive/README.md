# Arrays Interactive

A collection of array manipulation examples with GUI components.

## Topics Covered

- Array creation and initialization
- Searching (linear and binary)
- Sorting methods
- Insertion and deletion
- 2D arrays

## Key Files

- `InteractiveArray.java` - Main GUI application
- `ArraySortMethods.java` - Various sorting algorithms
- `ArraySearch.java`, `ArraySearchCeilModified.java`, `ArraySearchObjects.java` - Search implementations
- `ArrayCreate.java` - Array creation
- `ArrayPredictor.java` - Array prediction
- `ArrayTime.java` - Timing operations
- `ArrayChooseMostEfficient.java` - Efficiency comparison
- `InsertKey.java`, `DeleteKey.java` - Key-based operations
- `Array_2D_Average.java`, `Array_2D_Lengths.java`, `Array_2D_Dice.java` - 2D array examples
- `MyFrame.java`, `MyFrame2.java`, `ButtonClickListener.java`, `InsertKeyListener.java` - GUI components

## Usage

Run `InteractiveArray.java` for the GUI, or individual files for specific examples.
