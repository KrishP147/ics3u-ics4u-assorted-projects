# OOP Concepts

Examples demonstrating object-oriented programming principles in Java.

## Topics Covered

- Classes and objects
- Constructors
- Encapsulation (getters/setters)
- Inheritance
- Polymorphism

## Classes

- `MyFirstOOP.java` - Introduction to OOP
- `Message.java` - Message class example
- `Student.java`, `Employee.java` - Basic class examples
- `Circle.java`, `CircleTest.java` - Geometry with OOP
- `Animal.java`, `Cat.java`, `DogSimple.java` - Inheritance example
- `Account.java`, `ChequingAccount.java`, `SavingsAccount.java`, `SavingsAccountAvishan.java` - Banking hierarchy
- `TimeTravel.java` - Time travel example
- `ComputerLab.java` - Computer lab example

## Usage

Run individual files to see specific OOP concepts in action.
